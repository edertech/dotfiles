
#+TITLE: VSCodium 


[[https://vscodium.com][VSCodium]] is a community-driven, freely-licensed binary distribution of Microsoft’s editor VS Code.
Here I have my customized settings and key shortcuts to make VSCodium behave like [[https://github.com/doomemacs][DOOM Emacs]] using the vim and whichkey plug-ins.
